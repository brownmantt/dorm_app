## 保管場所マスタ一覧

**インターフェース名称：** 保管場所マスタ一覧
**機能説明：** 保管場所マスタの一覧をページング取得する
**インターフェースURL：** `/storage-locations`
**リクエスト方式：** GET

---

### 機能説明
保管場所名による部分一致検索を行い、名称昇順で一覧を返却する。備品保管登録画面の保管場所コンボボックスは本 API から取得したデータを表示する。

---

### リクエストパラメータ
```json
{
  "name": "倉庫",
  "page": 0,
  "size": 20
}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| name | string | いいえ | 保管場所名（部分一致） | 倉庫 |
| page | integer | いいえ | ページ番号（0 始まり） | 0 |
| size | integer | いいえ | 1 ページ件数 | 20 |

---

### レスポンスパラメータ
```json
{
  "content": [
    {
      "storageLocationId": "SL202606300001",
      "name": "本社倉庫1階",
      "createdAt": "2026-06-30T10:00:00",
      "updatedAt": "2026-06-30T10:00:00"
    }
  ],
  "totalElements": 3,
  "totalPages": 1
}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| content | array | はい | 保管場所一覧 | - |
| content.storageLocationId | string | はい | 保管場所 ID | SL202606300001 |
| content.name | string | はい | 保管場所名 | 本社倉庫1階 |
| content.createdAt | string | はい | 作成日時 | 2026-06-30T10:00:00 |
| content.updatedAt | string | はい | 更新日時 | 2026-06-30T10:00:00 |
| totalElements | integer | はい | 総件数 | 3 |
| totalPages | integer | はい | 総ページ数 | 1 |

---

## 保管場所マスタ登録

**インターフェース名称：** 保管場所マスタ登録
**機能説明：** 保管場所マスタを新規登録する
**インターフェースURL：** `/storage-locations`
**リクエスト方式：** POST

---

### 機能説明
保管場所名の重複をチェックし、保管場所 ID（`SL` プレフィックス）を自動採番して登録する。管理者のみ実行可能。

---

### リクエストパラメータ
```json
{
  "name": "本社倉庫3階"
}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| name | string | はい | 保管場所名（最大 100 文字、論理削除レコードを除き一意） | 本社倉庫3階 |

---

### レスポンスパラメータ
```json
{
  "storageLocationId": "SL202606300004",
  "name": "本社倉庫3階",
  "createdAt": "2026-06-30T10:00:00",
  "updatedAt": "2026-06-30T10:00:00"
}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| storageLocationId | string | はい | 保管場所 ID | SL202606300004 |
| name | string | はい | 保管場所名 | 本社倉庫3階 |
| createdAt | string | はい | 作成日時 | 2026-06-30T10:00:00 |
| updatedAt | string | はい | 更新日時 | 2026-06-30T10:00:00 |

---

## 保管場所マスタ更新

**インターフェース名称：** 保管場所マスタ更新
**機能説明：** 保管場所マスタを更新する
**インターフェースURL：** `/storage-locations/{id}`
**リクエスト方式：** PUT

---

### 機能説明
指定 ID の保管場所名を更新する。保管場所名の重複をチェックする。管理者のみ実行可能。

---

### リクエストパラメータ
```json
{
  "name": "本社倉庫1階（改）"
}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| id | string | はい | 保管場所 ID（パス） | SL202606300001 |
| name | string | はい | 保管場所名（最大 100 文字） | 本社倉庫1階（改） |

---

### レスポンスパラメータ
```json
{
  "storageLocationId": "SL202606300001",
  "name": "本社倉庫1階（改）",
  "createdAt": "2026-06-30T10:00:00",
  "updatedAt": "2026-06-30T11:00:00"
}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| storageLocationId | string | はい | 保管場所 ID | SL202606300001 |
| name | string | はい | 保管場所名 | 本社倉庫1階（改） |
| createdAt | string | はい | 作成日時 | 2026-06-30T10:00:00 |
| updatedAt | string | はい | 更新日時 | 2026-06-30T11:00:00 |

---

## 保管場所マスタ削除

**インターフェース名称：** 保管場所マスタ削除
**機能説明：** 保管場所マスタを論理削除する
**インターフェースURL：** `/storage-locations/{id}`
**リクエスト方式：** DELETE

---

### 機能説明
指定 ID の保管場所を論理削除する。備品保管（`equipment_storage`）で参照中の保管場所は削除不可。管理者のみ実行可能。

---

### リクエストパラメータ
```json
{}
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| id | string | はい | 保管場所 ID（パス） | SL202606300001 |

---

### レスポンスパラメータ
```json
null
```

| パラメータ名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| body | null | はい | 成功時は空レスポンス | null |
