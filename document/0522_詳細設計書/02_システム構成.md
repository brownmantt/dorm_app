# 2. システム構成

> 親文書: [README.md](./README.md)

## 2.1 論理構成

```text
[ブラウザ (React SPA)]
        │ HTTPS / JSON
        ▼
[API Gateway / Reverse Proxy]  ※本番化時
        │
        ▼
[寮管理 API (Spring Boot)]
   ├─ Presentation  (Controller)
   ├─ Application     (UseCase / Service)
   ├─ Domain          (Entity / 業務ルール)
   └─ Infrastructure  (Repository / Excel・CSV / Log / 印刷 / 外部郵便番号 API)
        │
        ▼
[PostgreSQL]
```

## 2.2 パッケージ構成（バックエンド）

```text
com.example.dormitory
├── presentation      # REST Controller, DTO, バリデーション
├── application       # *Service, *UseCase（トランザクション境界）
├── domain
│   ├── dormitory     # 寮・部屋・責任者
│   ├── affiliation   # 所属マスタ
│   ├── allocation    # 寮割カレンダー・印刷
│   ├── residence     # 入退寮・変更履歴・初回利用日
│   ├── fee           # 寮費算定
│   ├── equipment     # 備品
│   ├── vacancy       # 空き室
│   └── importexcel   # Excel/CSV 取込・出力
└── infrastructure    # JPA Repository, 操作ログ, ファイルストレージ
```
