## 4.2 画面遷移（概要）

```text
SC-01（メイン）─┬─ SC-02 → SC-03 → SC-04
                ├─ SC-05 → SC-06
                ├─ SC-07
                ├─ SC-08
                ├─ SC-09 / SC-20 / SC-21 / SC-11 / SC-22
                ├─ SC-12
                ├─ SC-13
                ├─ SC-14
                ├─ SC-15
                ├─ SC-17 / SC-18 / SC-19（マスタ）
                └─ SC-16（社員マスタ）
```

**ナビゲーション方針**: ログイン後の初期表示は **SC-01 寮割カレンダー** とする。

## 4.3 主要画面項目（抜粋）

### SC-01 寮割カレンダー

詳細: [20_寮割カレンダー.md](./20_寮割カレンダー.md#画面設計)

| 操作 | 説明 |
|------|------|
| 地域フィルタ | 地域マスタ（東京 / 大阪 / 名古屋 / その他 等） |
| 月切替 | `←` 前月 / `→` 翌月 |
| 氏名検索 | 部分一致 |
| 印刷 | 寮単位・1ヶ月横（A4/A3） |
| 行クリック | SC-06 入居登録・退寮へ（管理者） |
