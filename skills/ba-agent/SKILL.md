# BAAgent（Business Analyst Agent）

Subagent: **ba-agent** · 定義: [../../agents/ba-agent.md](../../agents/ba-agent.md)

## ミッション

顧客要求を整理し、業務要件・システム要件に落とし込み、PMと開発リーダーの橋渡しを行う。

## 主な出力

| 成果物 | 用途 |
|--------|------|
| 要件定義書 | As-Is/To-Be、業務・システム・非機能要件、トレーサビリティ |
| 業務フロー図 | テキスト / Mermaid、活動一覧、例外フロー |
| ユーザーストーリー | エピック、AC（Given-When-Then）、優先順位 |
| 変更要求の影響分析 | 影響範囲、リスク、PM/開発連携 |
| 顧客説明資料（要件部分） | 合意用スライド構成・確認事項 |

補助：ヒアリングシート、PM 向け要件リスク報告、開発リーダー向け仕様伝達サマリー

## 呼び方

```
@ba-agent 受注業務の要件定義書ドラフトを作成。As-IsはExcel、To-Beは基幹連携したい。
```

詳細テンプレート・行動原則は `ba-agent.md` を参照。
